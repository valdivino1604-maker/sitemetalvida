# Metal Vida Engenharia

Site profissional estatico para publicar no GitHub Pages.

## Arquivos principais

- `index.html`: pagina principal do site.
- `assets/styles.css`: visual do site.
- `assets/script.js`: menu, botoes de WhatsApp e formulario.
- `assets/logo-metalvida.png`: logo da empresa.
- `assets/favicon.svg`: icone do site.
- `CNAME`: dominio personalizado `www.metalvida.com.br`.
- `.nojekyll`: evita que o GitHub Pages altere a publicacao.

## Como publicar no GitHub Pages

1. Crie um repositorio no GitHub, por exemplo `metalvida-site`.
2. Envie todos os arquivos desta pasta para o repositorio.
3. No GitHub, abra `Settings`.
4. Clique em `Pages`.
5. Em `Build and deployment`, escolha `Deploy from a branch`.
6. Em `Branch`, escolha `main` e pasta `/root`.
7. Clique em `Save`.

Depois disso, o GitHub vai publicar o site. Se o dominio estiver apontado para o GitHub Pages, ele abre em `www.metalvida.com.br`.
